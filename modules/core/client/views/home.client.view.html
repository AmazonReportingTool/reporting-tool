Amazon Product Reporting Tool

This reporting tool will allow Amazon Prime sellers to be able to easily keep track of their profit margins and other business metrics for each one of their products. It has an easy to use User Interface that allows users to easily navigate and customize the reporting tool.

Featuring:

  • Sku Report
  • Brand Report
  • Profit Margins
  • Automatically adjusts current FBA fees
  • FBA Percentage
  • Customizable Timeframes
  • Customizable Profit Margin Threshold
  • Clearly Displays those products that are in danger of not meeting the Threshold
